## Installation

Clone the repository:

```bash
git clone https://github.com/Jagappa/JagappaM_Portfolio.git
```

Navigate to the project:

```bash
cd JagappaM_Portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Build

```bash
npm run build
```
